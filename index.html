Visual Studio Code is a source-code editor made by Microsoft for Windows , Linux and macOS . Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code

refactoring, and embedded Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add additional functionality.

The console.log() method in HTML is used for writing a message in the console. It indicates an important message during testing of any program. The message is sent as a parameter to the console.log() method.

To open the console in Chrome, use this keyboard shortcut: Cmd + Option + J (on a Mac) or Ctrl +Shift +J (on Windows).

As an alternative, you can right-click on the webpage and click "Inspect" to open the developer console.
